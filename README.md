# Portfolio Website

This is a personal portfolio website showcasing projects related to UI/UX design and web development. It serves as a digital showcase of my skills, projects, and design philosophy.

### Features

* Simple and clean UI design
* Smooth scrolling navigation
* Project showcase with separate pages
* Responsive layout
* Basic animations (fade-in effects)

### Projects

* **EcoTrack Mobile App** – A sustainability tracking application
* **Nova Analytics Dashboard** – A data visualization dashboard
* **Lumina Smart Home** – A smart home interface concept

### Technologies Used

* HTML5
* CSS3

### How to Run

1. Clone the repository
2. Open `index.html` in any web browser
