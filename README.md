## Project Structure

The project consists of three main components:

- **index.html**: The main structure of the website using semantic HTML5 tags like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`.
- **style.css**: A separate stylesheet that defines the layout, colors, and responsive behavior of the website.
- **README.md**: Documentation for the project.

## Website Sections

1.  **Home (Hero Section)**: A welcoming introduction with the student's name and title.
2.  **About Me**: A short biography and a list of technical skills.
3.  **Projects**: A grid displaying three example project cards. Each card includes an image placeholder, a title, a description, and a button.
4.  **Contact**: Simple contact information including email and social media profile placeholders.
5.  **Footer**: A consistent footer with copyright information, quick links, and social media placeholders.

## Key Features

- **Semantic HTML**: Uses appropriate HTML tags for better accessibility and structure.
- **Responsive Design**: Includes CSS media queries to ensure the website looks good on both desktop and mobile devices.
- **Internal Navigation**: A horizontal navigation menu that uses anchor links (`#section-id`) to scroll smoothly between sections on the same page.
- **Clean Styling**: A modern, minimalist aesthetic using a simple color palette and card-style layout for projects.

## How to Use

1.  Open `index.html` in any modern web browser to view the portfolio.
2.  Customize the text, links, and images in `index.html` to match your own details.
3.  Modify the colors and styles in `style.css` if you want to personalize the design further.
